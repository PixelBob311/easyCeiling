	систему Windows - программу для открытия файла можно изменить. Для этого нужно нажать правой кнопкой мыши на  ярлык требуемого файла, выбрать пункт "открыть с помощью", 
	в появившемся окне выбрать предпочитаемую программу для открытия файла и внизу окна поставить галочку "Всегда использовать приложение для открытия файлов данного формата".
		Если в появившемся окне нет нужной программы для открытия файла - необходимо раскрыть спиок приложений полностью, затем нажать кнопку "выбрать другое приложение"
	и провести в папке Мой компьютер поиск ярлыка требуемого приложения. 
	
2)		Если задача вопрос заключается в удаленном доступе к другому компьютеру - можно воспользоваться приложением TeamViewer. Если вопрос заключается именно в создании 
	локальной сети м/у двумя компьютерами - здесь можно воспользоваться настройками сетевого подключения, создав локальную сеть, настроив службу DNS, присоединиться 
	к известному IP адресу с разницей в последней значащей группе. Еще для этого потребуется создание расшареной папки, чтобы пользователи могли к ней обращаться.
		Также, можно сделать выделенную сеть, воспользоваться методом туннелирования, а еще воспользоваться разными программами типа
	Hamachi, ZeroTier и тд. Если честно, у меня не хватит компетентности, чтобы сходу ответить на этот вопрос, либо же нужно уточнить, что конкретно хочет получить пользователь в итоге.
3)		
		Если папка C:\Users\User отсутствует - это может говорить о том, что пользователя под именем User в системе нет. Для того, чтобы узнать ваше имя в системе - нажмите "Пуск"
	 и под картинкой в правом верхнем углу будет указано ваше текущее имя в системе. Попробуйте проверить наличее папки C:\Users\"имя пользователя"
		Также, папка может отсуствовать, потому что ваша учетная запись не предпологает взаимодействие с этой папкой. Это может быть обусловленно политикой безопасности вашей
	компании. Подробности можно узнать у системного администратора вашей компании.
